# flutter-web-spa-animated-hello-world

## Description
A POC project for flutter with animation.

## Tech stack
- flutter

## Docker stack
- debian
- nginx

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
